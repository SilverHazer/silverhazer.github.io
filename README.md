# Mijn streamingdienst voor Tam en mam

Steps:

2. Installeer [Media Station X vanuit de LG app store](https://us.lgappstv.com/main/tvapp/detail?appId=464834)
3. Launch de Media Station X applicatie
   ![MSX Start Menu](https://raw.githubusercontent.com/bartche/msx/main/imgs/msxmain.jpeg)
5. Navigeer naar Settings → Validate Links → No
   ![MSX Validade Links Menu](https://raw.githubusercontent.com/bartche/msx/main/imgs/validadelinks.jpeg)
7. Navigeer naar Settings → Start Parameter → Setup
   ![MSX Start Parameters](https://raw.githubusercontent.com/bartche/msx/main/imgs/startparam.jpeg)
9. Set the security lock (for HTTPS mode) en enter: silverhazer.github.io <br>
Save en klaar!
