# silverhazer.github.io

Steps:

Create a Stremio account <br>
Install the Media Station X from the LG app store <br>
Launch the Media Station X application MSX Start Menu <br>
Navigate to Settings → Validate Links → No MSX Validade Links Menu	<br>
Navigate to Settings → Start Parameter → Setup MSX Start Parameters <br>
Set the security lock (for HTTPS mode) and enter: silverhazer.github.io <br>
Save it and you are done!
