# silverhazer.github.io

Steps:

Create a Stremio account
Install the Media Station X from the LG app store 
Launch the Media Station X application MSX Start Menu
Navigate to Settings → Validate Links → No MSX Validade Links Menu	
Navigate to Settings → Start Parameter → Setup MSX Start Parameters
Set the security lock (for HTTPS mode) and enter: silverhazer.github.io
Save it and you are done!
